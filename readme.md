# Steam 游戏封面墙

这是一个根据您的 Steam 游戏库游玩时长，动态生成一张充满艺术感的封面墙的工具。游戏时长越长，封面展示的尺寸就越大，并使用智能算法紧密地排列在一起，形成一面壮观的游戏墙。

## 🔒 隐私声明

本工具完全在您的本地计算机上运行，我们尊重并保护您的隐私。

-   **无数据上传**：本工具**不会**以任何形式收集、存储或上传您的 SteamID、API Key 或任何游戏库信息到任何远程服务器。
-   **本地存储**：您输入的 Steam API Key 和查询历史记录，仅使用浏览器的 `localStorage` 技术保存在**您自己的电脑**上，仅供您个人方便使用。
-   **直接通信**：所有对 Steam API 的请求都由您在本地运行的后台代理服务直接发起。数据流向为：`您的浏览器 -> 您的本地后台 -> Steam API`，不经过任何第三方。

## ✨ 主要功能

- **动态瀑布流布局**：根据游戏时长自动调整封面大小，并使用 CSS Grid 的 `dense` 算法智能填充空隙。
- **自定义 API Key**：支持输入并保存您自己的 Steam Web API Key，避免共享 Key 的速率限制。
- **查询历史记录**：自动保存最近 10 条成功的查询记录，方便一键重新加载。
- **一键下载封面墙**：集成 `html2canvas`，可以将生成的整面墙下载为一张高清 PNG 图片。

## 🚀 如何使用

这个项目分为两个部分：一个**本地后台代理服务**（用于解决跨域问题）和一个**前端页面**（您看到的主界面）。您需要先运行后台，再打开前端页面。

1. **确保您已安装 [Node.js](https://nodejs.org/)**。
```bush
  pnpm install
  或者 npm install
```

2. 先启动后端的代理服务
```bush
  pnpm start
  或者 npm start
```

3. 打开前端页面 `index.html` （直接双击用浏览器打开即可）。


## 🚀 现在您应该可以看到项目的主界面了。

1.  **(可选) 输入您的 API Key**：在 "API Key" 输入框中填入您的 Steam Web API Key，然后点击 “保存 Key”。您的 Key 会被保存在浏览器本地，下次打开无需重新输入。
2.  **输入 SteamID64**：在 "SteamID64" 输入框中填入您想查询的用户的 64 位 ID。
3.  **生成封面墙**：点击 “生成” 按钮，等待片刻，您的游戏封面墙就会出现。
4.  **使用历史记录**：成功的查询会自动出现在输入框下方，直接点击即可快速加载。
5.  **下载图片**：生成成功后，点击 “下载封面墙” 按钮，即可将所见即所得的封面墙保存为一张 PNG 图片。
